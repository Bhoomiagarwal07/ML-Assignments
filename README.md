## 🛠️ How to Use
Each assignment folder is self-contained:
1. Open the `.ipynb` file in Google Colab or Jupyter.
2. Run all cells top to bottom.
3. Read that folder's `README.md` for the dataset link, methodology, and results specific to that assignment.
